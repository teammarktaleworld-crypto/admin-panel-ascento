⚡ Getting Started
1️⃣ Clone Repository
git clone https://github.com/teammarktaleworld-crypto/Cabtale.git
cd Cabtale
2️⃣ Install Dependencies
npm install
3️⃣ Run Project
npm run dev
<p align="center"> 👉 Open <b>http://localhost:3000</b> </p>
